API Documentation

API Documentation is a guide to help you understand how to use the API. It provides a list of all the endpoints and the parameters that can be used with them. It also provides examples of how to use the API.

Endpoints

There are 2 kind of endpoints in the API:

  • Customer
  • Admin

Customer's Endpoints

  • GET All Faculty

    <endpoint>/getFaculty

    Get all the faculties in the database.

  • GET Faculty by acronym

    <endpoint>/getFaculty/:Acronym
